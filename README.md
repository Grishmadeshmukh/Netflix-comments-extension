# Netflix Timestamp Commenting Extension

This browser extension allows users to comment on specific timestamps while streaming on Netflix. Follow the steps below to activate and use the extension.

---

## Steps to Activate the Extension

1. **Enable Developer Mode in Chrome Extensions**
   - Open Google Chrome and navigate to the extensions page by entering the following in the address bar:
     ```
     chrome://extensions/
     ```
   - Toggle the **Developer mode** switch in the top right corner.

2. **Load the Extension**
   - Click the **Load unpacked** button on the extensions page.
   - Select the folder where your extension files are located.

3. **Refresh After Making Changes**
   - If you modify the extension's code (e.g., `manifest.json`, `popup.js`), you'll need to refresh it:
     - Go back to the extensions page and click the refresh icon next to your extension.
     - Reload the Netflix page to apply the changes.

4. **Check on Netflix**
   - Open Netflix and start a video.
   - The extension should now be active. Interact with the video to verify its functionality.

---

## Notes
- Ensure the folder contains all required files (e.g., `manifest.json`, `popup.js`, `popup.html`).
- Debug any issues using the browser's developer tools (`Ctrl + Shift + I` or `Cmd + Option + I` on Mac).
